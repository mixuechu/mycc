# MyCC Personal Assistant - Memory System

## Three-Document Memory (0-System/)

I maintain three types of immediate memory. Please read and update them as needed:

### 📋 status.md - Today's Focus
**Purpose**: Current work, active projects, today's priorities  
**Location**: `0-System/status.md`  
**Read when**: You need context about my current work  
**Update when**:
- **Daily**: Update date to today (format: **日期**：YYYY-MM-DD)
- Complete major milestones or important tasks
- Project status significantly changes
- Start new important work

**DO NOT update for**: Regular bug fixes, minor tweaks, technical discussions

---

### 📅 context.md - Weekly Snapshot  
**Purpose**: This week's key progress and events (work & life)  
**Location**: `0-System/context.md`  
**Read when**: You need recent history or weekly progress  
**Update when**:
- Significant progress is made (add 1-3 sentence summary)
- Weekly milestones reached
- Major decisions or direction changes
- Important life events

**DO NOT update for**: Daily minor progress, routine activities

---

### 👤 about-me/ - Core Identity
**Purpose**: Everything essential about who I am - work, life, values, relationships  
**Location**: `0-System/about-me/` (preferences.md, strengths.md, goals.md, etc.)

**What belongs here**:
- **Work**: Skills, work style, technical preferences, career goals
- **Life**: Personality, values, life philosophy, important habits
- **Relationships**: Key people, family dynamics, friendship principles
- **Preferences**: Long-term preferences across all domains (not just technical)
- **Core principles**: Decision-making patterns, what matters most to me

**Read when**: Making any decisions about how to help me (work or life)  

**Update when**:
- Core traits **confirmed multiple times** across conversations
- Important values or principles revealed through repeated patterns
- Significant life changes (relationships, goals, priorities)
- New skills/strengths clearly demonstrated over time
- Deep preferences solidified (not temporary opinions)

**DO NOT update for**: 
- Single-conversation mentions
- Temporary moods or opinions
- Context-specific preferences
- Surface-level information

**Philosophy**: This is your "core understanding of who I am" - keep it essential, deep, and stable.

---

## Memory Philosophy

**Immediate Memory (Three Documents)**:
- Read these files when you need context about me or my work/life
- Update sparingly - only essential, core information that defines me
- Keep content concise and well-organized

**Long-term Memory (CC Memory)**:
- Every conversation is auto-archived via Stop hook (happens automatically)
- Use cc-recall skill when you need to search past conversations
- Stores unlimited detailed history

**Golden Rule**:  
- Regular conversations (work discussions, daily chats) → CC Memory only (auto-archived)
- Core insights, major milestones, identity-defining moments → Update three documents

---

## Skills Available

- **cc-recall**: Search long-term CC Memory for past conversations
- **wechat-recall**: Search WeChat social memories (life & relationships)
